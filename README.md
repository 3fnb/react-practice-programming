## 개요

책 <b>실전 리액트 프로그래밍</b>을 통해 리액트의 핵심 개념을 익히고 깃허브를 통해 정리하기

## 진행방식

- 매주 한 챕터씩 읽는다.
- 매일매일 당일 공부진도를 슬랙에 공유한다.
- 주 1회 오프로 모임을 가진다.
- 공부 진도 중에서 각자 파트를 맡아 서로에게 개념을 설명해준다.
- 각자 맡은 파트 중 핵심개념에 대한 퀴즈를 준비해온다. (issue에 올린다.)
